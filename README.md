FimFictionAPI
=============
An API of many functions of http://FIMFiction.net/.

Dependencies:
-------------
- commons-lang-2.6
- htmllexer-2.1
- json-simple-1.1.1
- jsoup-1.6.3

All of them are on maven.

This is at the moment identical to yawkats version 851b9e70dee68d57a3b0fe6544e64295c8414467
